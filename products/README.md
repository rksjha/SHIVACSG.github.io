# Products Folder — How to Add Content

## Drop files here to publish them automatically

Supported file types:
- **PDF** — Product brochures, catalogs, spec sheets
- **Images** — JPG, PNG, WebP product photos
- **Word/Excel/PPT** — Product documents and presentations

## Steps
1. Copy your file into this `products/` folder
2. Push to GitHub (`git add . && git commit -m "Add product" && git push`)
3. GitHub Actions automatically rebuilds the gallery page
4. Your file appears live on the website within 2–3 minutes

## Naming Tips
- Use descriptive names: `SCG-Solar-Panel-Catalog-2024.pdf`
- Avoid spaces if possible (use hyphens instead)
- Files are displayed alphabetically
