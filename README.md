# Shiva Consultancy Group — Public Website

**Live URL:** https://rksjha.github.io/SHIVACSG.github.io/

Professional website for SCG — products, services, advertorials, media, and small apps for public distribution and social media sharing.

---

## How to Publish New Content (3 Steps)

1. **Drop your file** into the correct folder below
2. **Push to GitHub** (commit & push from GitHub Desktop or Terminal)
3. **Done** — the website auto-rebuilds and goes live in ~2 minutes

---

## Folder Guide

| Folder | What to put here |
|--------|-----------------|
| `products/` | Product brochures, catalogs, spec sheets (PDF, images) |
| `services/` | Service brochures, proposals, case studies (PDF, Word) |
| `advertorials/` | Marketing articles, flyers, press releases (PDF, images) |
| `small-apps/` | HTML tool files (calculators, forms) |
| `media/images/` | Photos for social media sharing (JPG, PNG, WebP) |
| `media/videos/` | Video files (MP4, MOV) |
| `media/pdfs/` | Standalone PDF documents |
| `Product and service profile/` | Agri/infrastructure scheme documents (PDF) |
| `assets/images/` | Site logo and banner images |

---

## Automation — What Happens Automatically

Every time you push files to GitHub:

1. **GitHub Actions** triggers automatically
2. **Gallery pages rebuild** — every new file appears as a card with View + Share buttons
3. **Site deploys** to GitHub Pages — live within 2 minutes
4. **Social sharing** buttons (WhatsApp, LinkedIn, Twitter) added to every page

**No manual editing needed.** Just drop files and push.

---

## One-Time GitHub Pages Setup

If GitHub Pages is not yet enabled:

1. Go to your repo: `github.com/rksjha/SHIVACSG.github.io`
2. Click **Settings** → **Pages**
3. Under **Source**, select **GitHub Actions**
4. Save — done. The next push deploys automatically.

---

## File Naming Best Practice

- Use hyphens instead of spaces: `SCG-Agri-Brochure-2024.pdf`
- Be descriptive — the filename becomes the card title on the website
- Avoid special characters other than hyphens and underscores
